Héritage des Classes

L'Héritage en JavaScript est un mécanisme qui permet à un class d'hériter les propriétés et les méthodes d'une autre classe, efin de réutiliser le code existant et de créer de hiérarchies de Classes

L'héritage implémente une relation de type "est-un" entre deux objets. Par exemple

            import { Person } from ".person.js"
            et
            class
