Les modules: JavaScript sont un mécanisme de regroupement et d'orgnaisation du code JavaScrpt. Ils permettent de découper un programme en plusieurs fichiers, appelés modules, qui peuvent être réutilisés et importés dans d'autres fichiers.

Pour créer un module JavaSript, on utilise le mot-clé export pour déclarer les éléments (fonction, variables, classes) que vous souhaitez rendre accessibles à d'autres modules.
Par exemple: 

                // Dans un fichier nommé "module.js"
                //Exporter une fonction 
                exporter function maFonction() {
                    // Code de la fonction
                }
        
                //Exporter une variable 
                export const maVariable = 42;
        
                // Exporter une classe 
                export class MaClasse {
                    // Code de la classe
                }

Ensuite, dans un autre fichier, nous pouvons utiliser le mot-clé "import" pour importer les éléments exportés depuis le module : 

                // Dans un autre fichier
        
                // Importer une fonction 
                import {maFonction} from './module.js';
        
                // Importer une variable 
                import {maVariable} from './module.js';
                
                // Importer une variable 
                import {MaClasse} from './module.js';
