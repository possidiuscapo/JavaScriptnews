Portée de variable

En javascript, la portée des variables determine où une varieble est accessible et utilsable dans votre code.La portée est définie par l'endroit où vous la déclaré et peut varier en fonction du type de déclaration de variable utilisée.

Il existe trois type de portée

Portée globale une variable déclarée en dehors de toutes les fonctions est une variable globale et peut être utilisée dans tout le code JS. Elle est accessible à la fois à l'intérieur et à l'extérieur des fonctions.

let globaleVariable = 10; function foo() { console.log(globaleVariable); //Accès à la variable depuis la fonction }

Portée function ou local une variable déclarée à l'intérieur d' une fonction à une portéelocale et n'est accessible qu'à entre la fonction. Elle est invisible en dehors de la fonction

function foo() { let localeVariable = 20; console.log(localeVariable); //Accès à la variable depuis la fonction }

Portée de bloc une variable déclarée à l'interieur des accolades "{}" avec let et const est accessible uniquement à l'intéreur de ce bloc

let globaleVariable = 10; function foo() { console.log(globaleVariable); //Accès à la variable depuis la fonction }
